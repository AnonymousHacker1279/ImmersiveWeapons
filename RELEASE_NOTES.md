This updates adds some new content as well as fixing bugs. It will be the last major release for MC 1.21.11.

### Feature Changes / Additions

- Redesigned Cloud Island structure
    - The structure has been completely rebuilt and is larger than before
    - Structure generation rates have been slightly reduced
    - Ventus Shard drop rates have been reduced to account for more ore present in the island
    - Ventus Smithing Templates drop from a vault
- Ventus Staff changes
    - New textures for the staff and the staff core
    - Updated recipe for the staff
    - Staff Core now drops from a vault on the Cloud Island
- Added Mauls
    - A new weapon type
    - Slow-swinging but high-damaging
    - Each tier bypasses a certain amount of armor
- The Chinese translation has been updated

### Bugfixes

- Fix Molten/Tesla/Ventus/Astral spears using the wrong handle texture
- Fix the Iron Gauntlet's alternate model occasionally failing to load due to a race condition
