This major update ports to MC 1.21.10.

### Feature Changes / Additions

- IW copper items have been removed as vanilla now includes them
- Tiltros noise generation is now fully customized via datagenerators
  - This allows for a new floating island style generation that is much more vertical than before
  - Tiltros biomes have been reworked to better fit the new generation style
- Added a deepslate variant of the Rock Spider when spawning at or below Y=0
- Added a Mooglow entity that spawns in the Starlight Plains
  - Similar to the Mooshroom
- Redesigned Evil Eye flight goals to be more natural and less jittery
- Ventus armor now has a 15% fall damage reduction per armor piece, on top of the existing 10 blocks of safe fall distance
- Slightly increased part sizes on the Lava Revenant to make it easier to hit
- Added Wisp entities to the Tiltros Wastes
- Nerfed pike damage and attack speed

### Bugfixes

- Fix advancement window background being a missing texture
- Fix pike and gauntlet attributes not being set as expected
- Fix pixel color order in bullet impact particles
- Fix bulletproof stained glass items not being translucent
